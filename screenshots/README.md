# Dashboard screenshots

PNG screenshots of each of the four dashboard pages live here so reviewers can preview the report on GitHub without downloading the .pbix.

Suggested filenames:
- `01_executive_summary.png`
- `02_encounter_activity.png`
- `03_financial_claims.png`
- `04_geographic_demographic.png`

Export each page from Power BI Desktop with **File → Export → Export to PDF** and convert to PNG, or use **File → Export → Export as PowerPoint** and extract slides. Whichever path keeps the visuals legible.
